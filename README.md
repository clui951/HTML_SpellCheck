HTML Spell Checker

Bulk Spell Checking and Replacement for a Website


---------

<p>

####For Jar Executable:

<p>

####How to compile and execute from command line: 
From the src file, enter commands
* javac -cp '.:../*' HTML_SpellCheck.java
* java -cp '.:../*' HTML_SpellCheck
